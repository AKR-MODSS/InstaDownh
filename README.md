### Deploy VPS

#### Configuration:

```
put the API_ID, API_HASH, BOT_TOKEN,OWNER_ID,OWNER_USERNAME at Config.py

```


##### First step:

```sh
git clone https://github.com/riz4d/InstaDown
```

##### Second step:

```sh
cd InstaDown 
```

##### Third step:

```sh
pip install -r requirements.txt
```

##### Fourth step:

```sh
python3 main.py
```
